## OpenClaw — Utiliser un abonnement ChatGPT pour alimenter OpenClaw via Codex OAuth
- Source : YouTube Short "Use ChatGPT Subscription to power OpenClaw" — Prompt Circle AI
- URL : https://youtube.com/shorts/UyZuiLJL_nQ

### Concept :
On peut utiliser son abonnement ChatGPT (Plus/Pro/Business/Enterprise) comme provider de modèle dans OpenClaw via le flow OAuth Codex. Ça permet d'utiliser GPT-5.5 sans clé API séparée, en passant par l'authentification ChatGPT.

### Configuration :
- Provider : `openai-codex`
- Modèle recommandé : `openai-codex/gpt-5.5` (contexte 400K tokens)
- Note : GPT-5.3 n'est pas supporté dans OpenClaw (supprimé du catalogue). Utiliser GPT-5.5.

### Commandes CLI :
```bash
openclaw onboard --auth-choice openai-codex
openclaw models set openai-codex/gpt-5.5
```

### Config JSON (openclaw.json) :
```json5
{
  agents: {
    defaults: {
      model: { primary: "openai-codex/gpt-5.5" }
    }
  }
}
```
