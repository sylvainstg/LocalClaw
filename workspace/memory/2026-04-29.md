## Vidéo YouTube — L'univers n'est pas réel (mémorisée)
- **"Nobel Prize Just Given for Proving the Universe Isn't Real!"** — Tom Bilyeu (31 min)
- URL : https://youtu.be/CM4TXfpsWUg

### Concepts clés :

**Définitions fondamentales :**
- **Localité** : les objets ne peuvent affecter/être affectés que par leur environnement physique immédiat. L'information voyage à un coût (temps, énergie). La distance est une barrière.
- **Réalisme** : les objets ont des états définis qu'on les observe ou non. La chaise existe quand tu quittes la pièce.
- **Prix Nobel 2022** : décerné à Aspect, Clauser et Zeilinger pour avoir prouvé que l'univers n'est PAS localement réel.

**Expériences clés :**
1. **Double fente (Young, 1801)** — La lumière produit un pattern d'interférence = comportement ondulatoire
2. **Photon unique (1986)** — Même un photon à la fois produit le pattern d'interférence = le photon passe par les DEUX fentes simultanément (superposition)
3. **Effet observateur** — Quand on place un détecteur, le pattern disparaît. Le photon "sait" qu'il est observé et se comporte comme particule.
4. **Choix retardé (Wheeler/2007)** — La décision d'observer APRÈS le passage du photon change rétroactivement ce que le photon a fait = le présent modifie le passé
5. **Inégalités de Bell (Clauser 1972, Aspect 1980s, Zeilinger 2017)** — Les corrélations entre particules intriquées dépassent le seuil possible si des "variables cachées" existaient → Einstein avait tort

**L'analogie du moteur de jeu vidéo :**
- Un moteur de jeu ne rend que ce que le joueur observe (trop coûteux de tout calculer)
- Les objets hors vue n'existent que comme probabilités/code
- La distance dans un jeu est une illusion — tout est traité au même endroit dans le CPU
- L'univers fonctionne exactement de la même façon

**Argument de simulation (Nick Bostrom, 2003) :**
- 3 possibilités : (1) toutes les civilisations se détruisent avant, (2) aucune ne choisit de simuler, (3) nous sommes presque certainement dans une simulation
- Si UNE civilisation survit et fait des simulations → ratio simulations/réalité de base → probabilité d'être en réalité de base ≈ 0
- Elon Musk : probabilité d'être en réalité de base = 1 sur un milliard

**Conclusion :** L'univers rend à la demande, la distance est une illusion, rien n'est défini tant que le système n'en a pas besoin, le passé se résout rétroactivement à partir du présent.

## Vidéo YouTube — Vin pour un match du Canadien (mémorisée)
- **"Quel vin pour regarder un match du Canadien?"** — QUB radio, chronique Vino avec Mathieu Turbide (9 min)
- URL : https://youtu.be/JLmzQZjYMEg

### 3 recommandations :
1. **Bachelder PTG** (blanc, Niagara) — ~23$ SAQ. Assemblage pinot blanc + chardonnay. Sec, fruité, rondeur, léger beurré. Par Thomas Bachelder, ex-journaliste montréalais devenu vigneron. Parfait avec ailes de poulet piquantes/BBQ.
2. **Planeta Frappato** (rouge léger, Sicile) — Cépage frappato, fruité, épicé. Parfait avec la pizza (fromage, pepperoni).
3. **Monastrell espagnol** (rouge concentré) — Aussi appelé Mourvèdre en France. Plus riche, rond, confituré. Bon avec tout le junk food de match.

### Notes :
- PTG = "Passe-Tout-Grain" — expression française pour un assemblage de tout le vignoble
- Terme "minéralité" dans le vin = invention récente (années 2000), n'existait pas dans les livres classiques de sommellerie
- Le boisé des vins californiens vient du brûlage intérieur des barriques de chêne (scellant d'origine), popularisé par le critique Robert Parker dans les années 90
